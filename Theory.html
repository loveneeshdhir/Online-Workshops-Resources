HTML FORMS Syntax:
*Input tag The input tag creates interactive controls.The type attribute determines the type. *Button Tag The button that submits the form at the end *Placeholder Attribute Placeholder will show the given word in place of empty text box *Value Attribute It gives value to an entered field that will be submitted *Name Attribute We can give names to data so that we can retrieve them *Select Tag A select tag consists of option tags inside it that show the different options that are available for that dropdown menu. *Labels These are the captions given to fields in a form.
